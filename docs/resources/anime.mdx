# Anime

import ApiCodeBlock from '../../src/components/ApiCodeBlock';
import Highlight from '../../src/components/Highlight';
import ApiTryIt from '../../src/components/ApiTryIt';

export const endpoints = [
  { method: 'GET', uri: '/v1/anime/:id' },
  { method: 'GET', uri: '/v1/anime' },
  { method: 'GET', uri: '/v1/random/anime/:count/:nsfw' }
];

This is an object representing an Anime show.
The API allows you to retrieve individual Anime as well as a list of them using various filters.

<ApiCodeBlock title="Endpoints" items={endpoints} />

## The Anime object

### Attributes

---

#### id <Highlight bgColor="#8D46B8" color="white" content="integer" />

Unique identifier for an Anime.

---

#### anilist_id <Highlight bgColor="#8D46B8" color="white" content="integer" /> <Highlight bgColor="rgb(40, 42, 54)" color="white" content="nullable" />

***[AniList](https://anilist.co/)*** external unique identifier.

---

#### mal_id <Highlight bgColor="#8D46B8" color="white" content="integer" /> <Highlight bgColor="rgb(40, 42, 54)" color="white" content="nullable" />

***[MyAnimeList](https://myanimelist.net/)*** external unique identifier.

---

#### tmdb_id <Highlight bgColor="#8D46B8" color="white" content="integer" /> <Highlight bgColor="rgb(40, 42, 54)" color="white" content="nullable" />

***[TheMovieDatabase](https://www.themoviedb.org/)*** external unique identifier.

---

#### format <Highlight bgColor="#8D46B8" color="white" content="enum" />

The show's format destination.

```js title="Possible format enum values"
"TV": 0,
"TV_SHORT": 1,
"MOVIE": 2,
"SPECIAL": 3,
"OVA": 4,
"ONA": 5,
"MUSIC": 6
```

---

#### status <Highlight bgColor="#8D46B8" color="white" content="enum" />

The show's global release status.

```js title="Possible status enum values"
"FINISHED": 0,
"RELEASING": 1,
"NOT_YET_RELEASED": 2,
"CANCELLED": 3
```

---

#### titles <Highlight bgColor="#8D46B8" color="white" content="map of strings" />

A dictionary of the show's titles organized by localization.

---

#### descriptions <Highlight bgColor="#8D46B8" color="white" content="map of strings" />

A dictionary of the show's descriptions organized by localization.

---

#### start_date <Highlight bgColor="#8D46B8" color="white" content="date" /> <Highlight bgColor="rgb(40, 42, 54)" color="white" content="nullable" />

The show's global release date.

---

#### end_date <Highlight bgColor="#8D46B8" color="white" content="date" /> <Highlight bgColor="rgb(40, 42, 54)" color="white" content="nullable" />

The known show's global end date.

---

#### weekly_airing_day <Highlight bgColor="#8D46B8" color="white" content="enum" /> <Highlight bgColor="rgb(40, 42, 54)" color="white" content="nullable" />

The known show's episode release day.

```js title="Possible day enum values"
"Sunday": 0,
"Monday": 1,
"Tuesday": 2,
"Wednesday": 3,
"Thursday": 4,
"Friday": 5,
"Saturday": 6
```

---

#### season_period <Highlight bgColor="#8D46B8" color="white" content="enum" />

The season on which the show has been released.

```js title="Possible season_period enum values"
"WINTER": 0,
"SPRING": 1,
"SUMMER": 2,
"FALL": 3,
"UNKNOWN": 4
```

---

#### season_year <Highlight bgColor="#8D46B8" color="white" content="integer" /> <Highlight bgColor="rgb(40, 42, 54)" color="white" content="nullable" />

The year on which the show has been released.

---

#### episodes_count <Highlight bgColor="#8D46B8" color="white" content="integer" />

Number of episodes released for the show.

---

#### episode_duration <Highlight bgColor="#8D46B8" color="white" content="integer" /> <Highlight bgColor="rgb(40, 42, 54)" color="white" content="nullable" />

The show's episode average duration in minutes.

---

#### trailer_url <Highlight bgColor="#8D46B8" color="white" content="string" /> <Highlight bgColor="rgb(40, 42, 54)" color="white" content="nullable" />

External link to the show's trailer video.
Possible services:

* Youtube
* Dailymotion

---

#### cover_image <Highlight bgColor="#8D46B8" color="white" content="string" />

The show's cover image.

---

#### has_cover_image <Highlight bgColor="#8D46B8" color="white" content="bool" />

Indicates if the show has an a cover images associated with it.

---

##### cover_color <Highlight bgColor="#8D46B8" color="white" content="string" /> <Highlight bgColor="rgb(40, 42, 54)" color="white" content="nullable" />

The show's cover main color, in `HEX` format.

---

##### banner_image <Highlight bgColor="#8D46B8" color="white" content="string" /> <Highlight bgColor="rgb(40, 42, 54)" color="white" content="nullable" />

The show's banner image.

---

#### genres <Highlight bgColor="#8D46B8" color="white" content="array of strings" />

A collection of the show's associated genres.
You can find all possible values ***[here](https://api.aniapi.com/v1/resources/1.0/0)***.

---

#### sagas <Highlight bgColor="#8D46B8" color="white" content="array of objects" /> <Highlight bgColor="rgb(40, 42, 54)" color="white" content="nullable" />

A collection of the show's associated sagas.

```js title="Saga's object structure"
{
  // Map of strings
  // Contains the saga's title in various localizations
  "titles": {}

  // Map of strings
  // Contains the saga's description in various localizations
  "descriptions": {}

  // Integer
  // The saga's first episode number
  "episode_from": 0

  // Integer
  // The saga's last episode number
  "episode_to": 0

  // Integer
  // The saga's total episodes count
  "episodes_count": 0
}
```
---

#### sequel <Highlight bgColor="#8D46B8" color="white" content="integer" /> <Highlight bgColor="rgb(40, 42, 54)" color="white" content="nullable" />

The show's precedent Anime's unique identifier in story-line.

---

#### prequel <Highlight bgColor="#8D46B8" color="white" content="integer" /> <Highlight bgColor="rgb(40, 42, 54)" color="white" content="nullable" />

The show's successive Anime's unique identifier in story-line.

---

#### score <Highlight bgColor="#8D46B8" color="white" content="float" />

The show's global appreciation indicator. Minimum value is `0` and maximum is `100`.

---

#### nsfw <Highlight bgColor="#8D46B8" color="white" content="bool" />

Indicates if the show is marked as ***[NotSafeForWork](https://www.techopedia.com/definition/27847/not-safe-for-work-nsfw)***

---

#### recommendations <Highlight bgColor="#8D46B8" color="white" content="array of integers" /> <Highlight bgColor="rgb(40, 42, 54)" color="white" content="nullable" />

The show's recommended Anime's unique identifier because similar. Ordered by descendent rating.

---

## Retrieve a specific Anime

<ApiCodeBlock items={[endpoints[0]]} />

Retrieves an Anime show, based on its unique identifier.

### Parameters

No parameters.

### Returns

Returns an Anime object if a valid identifier was provided.

### Try it

export const retrieveAnimeParams = [
  { name: ':id', type: 'number', placeholder: ':id', value: '11' }
];

<ApiTryIt items={retrieveAnimeParams} uri={endpoints[0].uri} />

## Get a list of Anime

<ApiCodeBlock items={[endpoints[1]]} />

Returns a list of Anime objects.
The Anime are returned sorted by `score`, with the most popular Anime appearing first.

### Parameters

---

#### title <Highlight bgColor="#8D46B8" color="white" content="string" /> <Highlight bgColor="rgb(40, 42, 54)" color="white" content="optional" />

A case-insensitive pattern filter on the list based on the `titles` field values.

---

#### anilist_id <Highlight bgColor="#8D46B8" color="white" content="integer" /> <Highlight bgColor="rgb(40, 42, 54)" color="white" content="optional" />

A filter on the list based on the `anilist_id` field value.

---

#### mal_id <Highlight bgColor="#8D46B8" color="white" content="integer" /> <Highlight bgColor="rgb(40, 42, 54)" color="white" content="optional" />

A filter on the list based on the `mal_id` field value.

---

#### tmdb_id <Highlight bgColor="#8D46B8" color="white" content="integer" /> <Highlight bgColor="rgb(40, 42, 54)" color="white" content="optional" />

A filter on the list based on the `tmdb_id` field value.

---

#### formats <Highlight bgColor="#8D46B8" color="white" content="array of enums" /> <Highlight bgColor="rgb(40, 42, 54)" color="white" content="optional" />

A filter on the list based on the `format` field value.

---

#### status <Highlight bgColor="#8D46B8" color="white" content="enum" /> <Highlight bgColor="rgb(40, 42, 54)" color="white" content="optional" />

A filter on the list based on the `status` field value.

---

#### year <Highlight bgColor="#8D46B8" color="white" content="integer" /> <Highlight bgColor="rgb(40, 42, 54)" color="white" content="optional" />

A filter on the list based on the `season_year` field value.

---

#### season <Highlight bgColor="#8D46B8" color="white" content="enum" /> <Highlight bgColor="rgb(40, 42, 54)" color="white" content="optional" />

A filter on the list based on the `season_period` field value.

---

#### genres <Highlight bgColor="#8D46B8" color="white" content="array of strings" /> <Highlight bgColor="rgb(40, 42, 54)" color="white" content="optional" />

A case-sensitive pattern filter on the list based on the `genres` field values.

---

#### nsfw <Highlight bgColor="#8D46B8" color="white" content="bool" /> <Highlight bgColor="rgb(40, 42, 54)" color="white" content="optional" />

A filter on the list which excludes Anime classified as **Not Safe For Work**.

---

#### with_episodes <Highlight bgColor="#8D46B8" color="white" content="bool" /> <Highlight bgColor="rgb(40, 42, 54)" color="white" content="optional" />

A filter on the list which excludes Anime without episodes available.

---

### Returns

Returns an array of Anime objects with a size based on the filter provided.

### Try it
